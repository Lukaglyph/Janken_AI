Janken_AI_v2.4
概要
JavaScriptで構築されたじゃんけんエンジンです。 ユーザーの行動パターンを10回ごとに動的に学習し、最適な対抗策をリアルタイムで生成します。

操作方法

https://lukaglyph.github.io/Janken_AI/ を任意のブラウザ（Chrome/Safari推奨）で開く。

下部のコントロールパネルから任意の手を選択。

10回ごとの Update Sequence（赤色のアラート）で学習完了を確認。

技術スタック
Frontend: HTML5, CSS3 (Modern Flex/Grid)

Engine: Pure JavaScript (ES6+)

Analysis: Markov-chain-like frequency analysis
